= User Agent

User agent parser.

== Example

  agent = UserAgent.new 'Mozilla/5.0 (Windows; U; Windows NT 6.0; en-us) AppleWebKit/531.9 (KHTML, like Gecko) Version/4.0.3 Safari/531.9'
  agent.name    # => :Safari
  agent.version # => '4.0.3'
  agent.engine  # => :webkit
  agent.os      # => :'Windows Vista'
  agent.engine_version  # => '531.9'

== Supported Agents

* Safari
* Chrome
* Firefox
* Opera
* IE
* Konqueror
* PS3
* PSP
* Wii
